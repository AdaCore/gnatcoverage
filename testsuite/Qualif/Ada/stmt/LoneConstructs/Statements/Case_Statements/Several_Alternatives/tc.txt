Check that if a CASE statement is executed  several times, then the statement
itself and all the component statements of only those alternatives that have
been executed are reported as covered, and the component statements of all
the other alternatives  are reported as uncovered. The test drivers execute
the functional code in various ways, each case statement except the case
statements from the elaboration code in the body of the package
More_CASE_Statements are executed several times.
