If a declaration is not elaborated as part of the program execution,
then it shall be reported as uncovered.

%(tstrategy-headline)s

The testing strategy considers the elements contained in chapter 3 
of the Ada Reference Manual which are relevant from a structural coverage
analysis perspective. It takes into account the following cases:

* Discrete Subtype Definitions.
* Object Declarations. 
* Subtype Indications.
* Variant Records.
