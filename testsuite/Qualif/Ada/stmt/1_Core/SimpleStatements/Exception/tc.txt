Check that if an exception is raised or a raise statement is executed, 
then all subsequent statements are reported as not covered.

Check that if no exception is raised, then all exception handlers are reported
as uncovered.

Check that if an exception is raised, then only the statements associated to
the correct handler are reported as covered.

Take into account implicit and explicit exception raise. Take into account
exception handlers with explicit exception names and including the "others"
choice. Take into account exceptions raised in functions, procedures and block
bodies.
